## Dashboard Overview

หน้าแดชบอร์ดจะแสดงข้อมูลสรุปของระบบ เช่น

- จำนวนสินค้าทั้งหมดในระบบ
- ตารางแสดงรายการสินค้า
- ระบบค้นหาสินค้า
- ปุ่มจัดการสินค้า

---

## ฟังก์ชันหลักของระบบ

### 1. เพิ่มสินค้าใหม่
ผู้ใช้งานสามารถเพิ่มสินค้าใหม่เข้าสู่ระบบ โดยกรอกข้อมูล เช่น

- ชื่อสินค้า
- ราคา
- รูปภาพสินค้า

---

### 2. แสดงรายการสินค้า

ระบบจะแสดงข้อมูลสินค้าในรูปแบบตาราง ประกอบด้วย

| ฟิลด์ข้อมูล | รายละเอียด |
|--------------|-------------|
| ลำดับ | หมายเลขรายการสินค้า |
| ชื่อสินค้า | ชื่อของสินค้า |
| ราคา | ราคาสินค้า (บาท) |
| รูปภาพ | รูปภาพสินค้า |
| จัดการ | ปุ่มแก้ไข ลบ และเพิ่มลงตะกร้า |

---

### 3. ค้นหาสินค้า

ผู้ใช้งานสามารถค้นหาสินค้าได้จากช่องค้นหา เพื่อให้สามารถค้นหาสินค้าได้รวดเร็วและสะดวก

---

### 4. แก้ไขสินค้า

สามารถแก้ไขข้อมูลสินค้า เช่น

- ชื่อสินค้า
- ราคา
- รูปภาพสินค้า

---

### 5. ลบสินค้า

สามารถลบสินค้าออกจากระบบได้ เมื่อสินค้านั้นไม่ต้องการใช้งานแล้ว

---

### 6. เพิ่มลงตะกร้า

ระบบมีปุ่มเพิ่มสินค้าลงตะกร้า เพื่อรองรับการนำสินค้าไปใช้งานในระบบขายสินค้า (Shopping Cart)

---

## เทคโนโลยีที่ใช้พัฒนา

- PHP
- MySQL / MariaDB
- HTML
- CSS
- Bootstrap
- JavaScript

## จุดเด่นของระบบ

- ใช้งานง่ายผ่านหน้า Dashboard
- จัดการสินค้าได้ครบ (CRUD)
- รองรับการอัปโหลดรูปภาพสินค้า
- มีระบบค้นหาสินค้า
- รองรับการต่อยอดเป็นระบบร้านค้าออนไลน์

---

## วัตถุประสงค์ของระบบ

- เพื่อใช้จัดการข้อมูลสินค้าอย่างเป็นระบบ
- ลดความผิดพลาดในการจัดเก็บข้อมูลสินค้า
- เพิ่มความสะดวกในการดูแลสินค้า
- รองรับการพัฒนาต่อยอดเป็นระบบขายสินค้าออนไลน์
